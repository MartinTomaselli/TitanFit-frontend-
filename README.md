# TitanFit Frontend

Frontend MVP de TitanFit.

## Tecnologías

- **React** - Librería de UI
- **Vite** - Build tool y dev server
- **Tailwind CSS** - Framework CSS utility-first
- **Supabase** - Backend y autenticación
- **Vercel** - Plataforma de deployment

## Instalación

```bash
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Build

Para crear una build optimizada para producción:

```bash
npm run build
```

## Deployment

Este proyecto está configurado para desplegarse en **Vercel**. Puedes conectar tu repositorio de GitHub directamente a Vercel para deployment automático.
